This plugin demonstrates how to use CSS values from the theme's theme.json in a plugin.

== Description ==

This plugin sets a home page with basic styling.

== Installation ==

1. Upload `holding` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
